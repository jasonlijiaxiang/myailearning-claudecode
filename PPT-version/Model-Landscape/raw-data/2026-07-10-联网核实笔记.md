# 2026-07-10 · Model-Landscape · 联网核实笔记

> 元信息：建模块前的时效性事实核实（流程第 2 步）。本模块是全库事实密度最高、最易过期的模块，
> 本笔记为讲义/书单唯一事实源；巡检时本模块优先核。所有价格与版本号均为 2026-07-10 口径。

## 一、闭源旗舰格局（西方）
- **OpenAI**：GPT-5.6 家族 2026-07-09 GA，三档：Sol（旗舰，$5/$30 每百万 token 输入/输出）、Terra（均衡，$2.50/$15，官方称约以 5.5 一半成本对齐 GPT-5.5）、Luna（快/低价，$1/$6）。上代 GPT-5.4 $2.50/$15 曾是"最佳全能"；GPT-5.4 Pro $30/$180。缓存：读折 90%，写按 1.25×。消费端统治：ChatGPT 约 74% 消费份额、日均 25 亿+ prompts。
- **Anthropic**：Claude 5 家族 = Fable 5（Mythos 级新档位，位于 Opus 之上；1M 上下文/128K 输出/自适应思考；出口管制风波后 2026-07-01 恢复全球访问）+ Mythos 5（仅限量开放）；Opus 4.8（企业/agentic 编码主推）；Sonnet 5（近 Opus 性能，尝鲜价 $2/$10 至 2026-08-31，之后 $3/$15）；Haiku 4.5。企业侧第一：Menlo 口径企业 LLM 支出份额 Anthropic 40% / OpenAI 27% / Google 21%；Claude Code 年化 $1B、企业编码市场 54%。
- **Google**：Gemini 3.1 Pro（推理与长上下文/视频王者，ARC-AGI-2 领先，旗舰中 API 最低价）；Flash / Flash-Lite（$0.10/$0.40，最便宜闭源档）；Live API 见 mm-voice-realtime。
- **xAI**：Grok 4.3（$1.25/$2.50、缓存输入 $0.20，比上代降 40–60%；AA 智能指数 53，性价比 Pareto 前沿定位）；2026-06-15 上 Bedrock（第三家独立实验室）；SOC2/HIPAA/GDPR。
- **Meta 转向**：Behemoth（2T）实质搁置未发布；2026-04-08 发 Muse Spark——首个闭源 API-only 前沿模型；Llama 4 Scout（109B/激活 17B，10M 上下文）与 Maverick（400B/17B，1M）成为"最后的开放版本"。

## 二、开放权重格局（中国主导）
- 开源榜首几乎全是中国队：DeepSeek V4 Pro（综合 87 第一；SWE-bench Verified 80.6% 追平闭源前沿；MIT）、GLM-5.1（83，MIT；GLM-5 首个完全用华为昇腾训练的前沿模型、零英伟达）、Kimi K2.6（81；Artificial Analysis 指数 54 = 开源第一/总榜第四；agentic 编码强项）、Qwen3.5 397B（77；Qwen3.5 122B/激活 10B 超 GPT-5-mini；家族尺寸谱系最全）。
- 开源与闭源差距收敛到个位数百分点（多数基准）；DeepSeek V4 Flash $0.14/$0.28 锚定价格地板。
- 西方开放权重：Llama 4（生态最深、上下文最长）；Mistral Large 3 / Small 4 已转 Apache 2.0；Gemma 4（26B MoE 激活约 4B；E2B 2.3B 有效参数 4GB 内存可跑）。

## 三、中国格局与豆包
- 国产四强（2026-06 口径）：GLM 5.2（智谱，企业云生态；昇腾全国产训练）、Kimi K2/K2.5（长文与 agent swarm，最多 100 并行子智能体）、Qwen3（尺寸最全 0.6B–397B、多语言）、DeepSeek V4（API 比 GPT-o 系便宜 20–50 倍）——各自至少一个主流基准超越国际闭源。
- **豆包（字节）**：Doubao 2.1 Pro（2026-06-23 发布，Coding/Agent/VLM"质变点"；¥6/百万输入、¥30/百万输出、缓存命中 ¥1.2）；2.1 Turbo = Pro 半价；Doubao-Seed-Evolving（Coding/Agent 专线，月更 2–4 次）；Doubao-Seed-2.0-lite（2026-05，家族首个原生统一视频/图/音/文全模态）；累计 tokens 调用 180 万亿+（火山口径）。定位：to C 免费日常助手 + to B 方舟 MaaS。
- 选型口径（社区共识）：低成本推理→DeepSeek；中文长文→Kimi；多语言/尺寸谱系→Qwen；企业云生态→GLM/通义；免费助手/字节生态→豆包。

## 四、许可证
- OSI 的 OSAID v1.0（2024-10）：要求开放训练数据信息/架构/训练代码——按此定义 Llama/Gemma/Qwen 都只是 open weight 不是 open source。
- 宽松名单（法务安全）：DeepSeek 系 MIT、GLM-5.1 MIT、Qwen3 系 Apache 2.0（0.6B–235B 全谱）、Mistral Large 3/Small 4 Apache 2.0、Gemma 自家许可。
- Llama Community License：>7 亿 MAU 需另谈授权 + 欧盟限制条款——大厂与欧企要过法务。

## 六、小模型与端侧
- 三大部署家族：Phi-4（含 multimodal 单模型视觉+音频）、Gemma 3/4（E2B 4GB 内存可跑）、SmolLM-3；Mistral Small 4（2026-03，Apache 2.0）。
- sub-10B 常规超越 2024 版 GPT-4（合成数据+蒸馏+架构精修）；有案例：精调 2.6B 有效参数在定向企业推理任务上超 DeepSeek-R1 671B。Apple 端侧 3B 常驻系统。

## 七、推理模型与能力轴
- 全部头部实验室均出推理模型；快慢混合思考成主流；预算控制：Anthropic budget_tokens / Gemini thinkingBudget；学界分 L1 可控 vs L2 自适应（arXiv 2507.02076 综述）。
- Overthinking 陷阱：超过临界思考预算后准确率稳定下降（arXiv 2506.04210）——"多想≠更好"。
- 上下文窗口谱系：128K（常规）→ 1M（Claude Fable 5 / Llama 4 Maverick / Gemini）→ 10M（Llama 4 Scout）。

## 八、选型方法论（企业实践）
- 多模型组合是 2026 企业默认解：Walmart Code Puppy 按成本/延迟/任务适配在 GPT/Claude/Gemini 间动态路由；ServiceNow 2026-01 同时与 OpenAI、Anthropic 签多年协议（按任务选模型）。
- 三层路由共识：最难推理→前沿闭源；高量明确任务→便宜档/开放权重；最敏感数据→自托管开放权重。
- Gartner：2028 年 70% 多模型组织将用 AI 网关（2024 年 <5%）——落地件在 ai-gateway 模块。
