AI-Infra-Compute · 算力硬件与网络

AI 基础设施的硬件底座：GPU/加速器到底是怎样一台机器（SM、HBM、Tensor Core）、显存与算力怎么算账、芯片选型格局（NVIDIA 各代 / TPU / 国产芯片），以及把上万张卡连成一台"超级计算机"的互联网络（NVLink / InfiniBand / RoCE）、集群拓扑、高性能存储与数据管线。面向售前技术，讲清这些硬件为什么这样搭、钱花在哪、选型时怎么取舍。
