基础层 · Foundation Layer

Prompt Engineering

和大模型对话的「接口层」：同一个模型，提示词写得好不好，效果可以天差地别。这是最便宜、最快见效的能力杠杆——不改一行权重、不搭一套检索，就能把模型已有的能力调出来。面向售前技术，讲清「怎么写、为什么这么写、什么时候该换成 RAG / 微调」，是 LLM、Agent、RAG 各模块共同的前置。
