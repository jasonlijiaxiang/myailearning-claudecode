LLM 原理

大模型的发动机舱：Transformer 架构与注意力机制（Attention）——模型凭什么理解上下文、凭什么能并行训练，上下文窗口、推理成本与各家价格差又是从哪来的。面向售前技术，讲清原理与取舍，是上层 RAG、Agent、MCP 模块共同的地基。
