## 真实事件 / 攻击（案例弹药）
- **EchoLeak（CVE-2025-32711）**：针对 Microsoft 365 Copilot 的首个真实"零点击"间接提示注入，CVSS 9.3；经 Markdown 图片渲染零点击外泄；一封精心构造邮件即可远程未授权窃取数据。来源：arXiv 2509.10540 / Sentra。
- **在野化**：2026-03 Unit 42 记录首批大规模在野间接注入（广告审核绕过、系统提示词泄露，22 种载荷工程手法）；CrowdStrike 报告 2025 影响 90+ 组织；Google 威胁团队 11 月→2 月在野间接注入 +32%；Bedrock agent 记忆投毒可跨会话存活；MCP 工具描述可完全接管编码 agent。来源：unit42.paloaltonetworks.com / helpnetsecurity 2026-04。
- **共识**：OpenAI / Anthropic / Google DeepMind 2025 均承认——现有 LLM 架构下提示注入无法被彻底根治，需在模型外用确定性策略兜底。

## 防护 · 护栏工具格局（2026）
- 开源：**NeMo Guardrails**（NVIDIA，Colang DSL，5 阶段管线，<100ms）/ **Llama Guard**（输入输出分类）/ **Guardrails AI**（50+ validator，结构化输出）/ **LLM Guard**。
- 云托管：**AWS Bedrock Guardrails**（6 类策略：内容过滤、拒绝主题、词过滤、敏感信息过滤、上下文接地检查、Automated Reasoning 检查）/ **Azure AI Content Safety**（Azure AI Foundry，含 Prompt Shields）/ **Lakera Guard**（实时 AI 防火墙，单 API 输入输出）。
- 架构级防御（2024–2026 收敛共识：在模型外用确定性策略兜底，而非训模型去拒绝）：
  - **Dual LLM 模式**（Simon Willison 2023）：Privileged LLM 管动作 / Quarantined LLM 处理不可信内容且无工具权、无持久态，仅经带类型的数据通道通信。
  - **CaMeL**（Google DeepMind 2025-03，Dual LLM 工程化）：Privileged LLM 生成受限 Python 程序，Quarantined LLM 无工具调用；目前评估最充分的架构级防御。arXiv 2506.08837《Design Patterns for Securing LLM Agents》给出 6 类模式（action-selector、plan-then-execute、dual LLM、code-then-execute、context-minimization、map-reduce）。
  - **最小权限 + 每动作授权 + HITL**：agent 只持当前任务所需最小权限，任务完成即回收。
- 红队：**Microsoft PyRIT**（Python 风险识别工具包，AI red team）/ Mindgard（持续 AI-SPM）等。
