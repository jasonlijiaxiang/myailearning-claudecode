## 阅读建议

1. **先读 #3 + #1(半天)**:客户问得最多的是"评估怎么起步、没标注数据怎么办"——#3 给"先定成功标准"的框架,#1 给"从错误分析长出评估集"的路子,两份合起来就是完整答法。
2. **被问"让 AI 评 AI 靠谱吗"之前,过一遍 #7**:三大偏差的实验数字(交换顺序漂移 >10%、长答案偏好 >90%)都在这篇里,答客户时有数据撑腰;#8 当索引备查,不必通读。
3. **客户拿榜单压你时,翻 #9**:"榜单第一"背后的私测择优机制,这篇讲得最透,是把话题从"刷榜"引向"贵司场景实测"的关键弹药。
4. **要做评估 demo 或 POC 验收方案,看 #2 + #4**:Anthropic 讲 Agent 验收分层,OpenAI 讲判官校准流程,照着搭就是一套能落地的 POC 评估方案。
5. **想系统补课再买 #6**:成书体系最完整,但前五份免费资料已覆盖日常售前所需,#6 可作为长期案头书;#10 仅在需要抠 RAG 指标定义时查。
