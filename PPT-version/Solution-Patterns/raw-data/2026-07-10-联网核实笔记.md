# 2026-07-10 · Solution-Patterns · 联网核实笔记

> 元信息：建模块前的时效性事实核实（流程第 2 步）。用于书单/讲义写作与 MANIFEST 时效性事实登记。

## 智能客服
- 头部厂商公布口径（注意都是厂商自报）：Fin（Intercom）平均解决率 51%、7000+ 客户样本另一口径 67%；Sierra 在 WeightWatchers 报约 70% 解决率 + 4.6/5 CSAT；Decagon 自报平均 deflection 80%。来源：fin.ai/learn 系列对比页、eesel/notch 2026 对比。
- 独立口径显著更低：Zendesk 企业中位数 41.2%（top quartile 58.7%）；厂商宣传与实测差 30–40 个百分点；deflection（挡走）≠ resolution（真解决），售前必须分清。来源：digitalapplied 2026 统计汇总。
- 采用度：Salesforce 报 66% 服务组织已跑 AI agent（2025 年 39%）；Gartner：91% CX 负责人有高管压力要上。

## 企业知识库 / AI 搜索
- Glean：2026-05 ARR 破 $300M（同比 +89%），估值 $7.2B（2025-06 Series F $150M，Wellington 领投）；产品形态 = 权限感知知识图谱（连邮件/文档/IM/工单/CRM）+ 企业搜索 + 助手 + Agent 平台；Glean Agents 年化 1 亿+ agent actions。来源：TechCrunch 2026-05-28、Glean 官方 press。

## AI Coding
- 市场 2026 年 $12.8B，85% 开发者用 AI 工具；预测 2032 年 $30.1B（27% CAGR）。
- 三分格局：Copilot 用户量第一（4.7M 付费）；Cursor 收入第一（$2B ARR、100 万+ 付费）；Claude Code 满意度第一（JetBrains 2026-04 调查 46% most-loved）。工作场景采用：Copilot 29% / Cursor 18% / Claude Code 18%（JetBrains 2026）。
- 70% 工程师同时用 2–4 个工具；主流组合 = IDE 内编辑（Cursor）+ 复杂任务终端 agent（Claude Code）。启示：客户方案按"补全 IDE + agentic 编码"双层设计。

## 数字人
- HeyGen：Avatar V（2026-04-08 发布，15 秒手机视频建照片级数字分身，10 分钟视频身份不漂移）；LiveAvatar（liveavatar.com 独立平台，WebRTC 实时双向交互数字人）。企业场景：7×24 客服、发布会主持、个性化销售。来源：HeyGen 官方 blog/help center。

## 营销内容生成
- 格局三角：Jasper（结构化高产量营销文案 + 品牌一致性：风格指南/语气示例/禁用词全局生效）；Canva Magic Studio（模板驱动日常素材，非专业设计团队）；Adobe Firefly Enterprise（商业安全定位：Adobe Stock+公版训练、企业客户 IP 赔付、可用品牌资产训练定制模型）。来源：genesysgrowth / technologyadvice 2026 对比、Adobe 官方。

## 跨场景（低代码平台，与 agent#agent-lowcode 同源）
- 中国智能体开发平台市场：火山引擎（HiAgent+扣子）以私有化 17.8% / 公有云 19.3% 双第一（IDC 口径，2026 报道）。
- 语音客服成本量级（与 mm-voice-realtime 同源）：级联管线 $0.0095–0.17/分钟；gpt-realtime 实测 $0.18–0.46/分钟（未缓存）、开缓存 $0.05–0.10/分钟。
