协议层 · Protocol Layer

A2A · 智能体间协议

A2A（Agent2Agent Protocol）：让不同厂商、不同框架、不同平台造出来的 AI Agent 能互相"发现对方、委派任务、同步进度、交换结果"的开放协议。与 MCP 是协议层两兄弟——MCP 接工具（纵向），A2A 接 Agent（横向）。面向售前技术，讲清"协议长什么样、怎么跑通一次协作、生产怎么落地、客户会怎么问"。

· 精选电子书书单：电子书书单.md（10 项，偏实践）✅
