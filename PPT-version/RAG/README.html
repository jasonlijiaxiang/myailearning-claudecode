应用模式层 · Application Patterns

RAG · 检索增强生成

检索增强生成（Retrieval-Augmented Generation）：给大模型外挂一个"随时可查、可更新、可溯源"的知识库。面向售前技术，讲清"为什么用、每一步怎么落地、上云对应什么服务、客户会怎么问"。原理五章（懂）+ 实践四章（跑）+ 进阶三章（广）。

本模块产出物：
· 讲义式 PPT：RAG-讲义.pptx（主力成品，128 页：原理 5 章 + 实践 4 章 + 进阶 3 章）
· 精选电子书书单：电子书书单.md（20 份，偏实践）
· 原始素材留痕：raw-data/
