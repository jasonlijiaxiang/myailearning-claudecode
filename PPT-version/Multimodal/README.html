应用模式层 · Application Patterns

Multimodal · 多模态

多模态（Multimodal）：让大模型不止「读写文字」，而是能看图、听声、读文档、看视频，还能反过来生成它们。面向售前技术，讲清「感知面全景、模态怎么拼、原生 vs 拼管线、SOTA 格局、选型落地、客户会怎么问」。位于「应用模式层」，与 RAG、Agent 同层。
