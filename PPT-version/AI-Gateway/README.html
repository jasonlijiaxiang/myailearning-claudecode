AI-Gateway · AI 网关

一旦你不止调用一个模型，"多对多"的接入、成本、安全就会失控。本模块把 AI 网关讲成一条运营主线：认清定位（统一入口 + 运营管控层）→ 统一接入 → 智能路由与容灾 → 按 token 管成本 → 挂上安全护栏 → 全链路可观测 → 连上 MCP 与 Agent → 会选型上云。面向售前技术，重取舍与落地，用"配电箱"类比讲活每个概念。
