# 2026-07-10 · AI-Ops · 联网核实笔记

> 元信息：建模块前的时效性事实核实（流程第 2 步）。所有口径为 2026-07-10。

## 一、OTel GenAI 语义约定现状
- 总体仍处 Development 状态、无稳定时间表；但 **gen_ai.client span（每次 LLM 往返一个 span）已于 2026 年初出实验期、达稳定**；gen_ai.agent span（每次 agent 调用）仍实验态但 2026 Q1 实践中已很稳。
- v1.41 规范已定义 agent / workflow / tool / model 四类 span + 延迟与 token 用量必备指标；多数 gen_ai.* 属性仍带 Development 稳定性标签（命名可能不经大版本变更）。
- 来源：opentelemetry.io semconv gen-ai / greptime 2026-05 解读 / OTel 官方博客 2026 GenAI observability。

## 二、工具格局（2026）
- **六平台锚定**：LangSmith（LangChain 系原生、框架绑定最深）、Langfuse（开源领袖、可自托管、OTel 框架无关；**2026-01 被 ClickHouse 收购**，开源线维护不变）、Arize Phoenix（ML 级统计功底：漂移检测/嵌入分析）、Helicone（drop-in 代理最简安装）、Datadog LLM Observability（Datadog 系企业默认）、Honeycomb（事件级深追踪）。
- Braintrust：定位"AI 产品质量管理系统"——观测与评估一体化，最慷慨免费档（1M span/月、10K eval）。
- AgentOps：agent 专用会话回放（time-travel debugging）、无限循环检测（识别递归思维模式、烧 token 告警/暂停）。
- APM 系 vs 专用系分工共识：Datadog/New Relic 管 token 成本与延迟够用；要输出质量评估、漂移检测、评分告警、非工程角色参与 → 必须配专用 AI 观测工具。
- 成本警示：RAG 管线遥测量是等价传统 API 的 **10–50 倍**；接入 AI 负载后 Datadog 账单普遍涨 40–200%——观测本身要做成本工程。

## 三、在线评估与反馈回流
- 共识漏斗：inline 检查 → judge 评审 → 人工纠正 → 离线学习；"评估指标正在变成生产监控指标"。
- 实践口径：**采样 5–10% 真实流量**异步打分（不加应用时延）；低分 trace 打上失败模式标签 → 进对抗测试集/黄金集——生产失败变回归覆盖（"dataset promotion"）。
- 闭环三件套：告警（急性回归）、评审队列（人工判断）、数据集晋升（失败→资产）。

## 四、漂移与静默退化
- 三类漂移：**供应商静默换 checkpoint**（上月测的模型≠这月答题的模型）、输出漂移（输入没变输出特征变）、性能漂移（语法对、事实错得更多）。
- 关键难点：基础设施指标全绿、系统"看起来正常"，质量在静默下滑——传统 APM 抓不到。
- 检测手段：嵌入空间距离、LLM-as-judge 持续打分、持续评估框架；逐调用记录语义相似度/成本/护栏违规。

## 五、发布管理（提示词/配置）
- 成熟形态：**版本注册表**（数据集+评估集+提示词版本+模型/参数+RAG 配置+工具+护栏全入册）→ 发布前评估门禁（自动检查+专家评审）→ 受控发布（环境/金丝雀/A|B/分阶段/发布说明/回滚）。
- 金丝雀口径：新提示词先切 **5–10% 流量**，观测质量分/错误率/延迟一个窗口再晋升；**与传统金丝雀的关键差异 = 必须在金丝雀流量上跑自动评估**，不能只看基础设施健康。
- 回滚纪律：回滚 = 改配置指针（生产 tag 指回上一已知好版本），不是重新部署；旧版本不可变、永远可用。

## 六、事故响应（AI 应用 on-call）
- AI 专属 runbook 四问：agent 答错了怎么办、被攻破了怎么办、卡死了怎么办、事后无法复盘推理链怎么办。
- **成本尖峰常是第一信号**：等人发现时，有写权限的 agent 可能已排队执行了补救动作——事故先于告警恶化。
- 必备机制：**急停开关**（从事故响应频道可触发、停任务不砸状态）；幻觉率作为可测运营指标（事实核查管线/一致性检查）；工具调用异常（意外函数调用/异常参数模式）进告警。
- 人机分级（HITL 三模型）：Human-in-the-Loop（人开车）/ Human-on-the-Loop（人监督、agent 有界行动）/ Human-out-of-the-Loop（人审计、agent 按策略行动）。
